Search_server - поисковой движок с поддержкой минус-слов (документы с этими словами не будут отображаться в результатах поисков). Ранжирование результатов происходит по TF-IDF.

Поддержаные функции:
·	Минус-слова
·	Многопоточный поиск
·	Разделение результатов поиска на страницы
·	Удаление дубликатов

Сборка
С помощью CMake собрать файл CMakeLists.txt, который находится в папке src.

Требования
C++17 и выше
